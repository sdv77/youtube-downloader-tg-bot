# youtube-downloader-tg-bot

## Magic script for easy setup Telegram Bot

This telegram bot downloads video as .mp3 files, this especially good for some music files

> On local machine script had proxy errors, but its only my issue, so you can try locally

Easy Setup:

`bash <(curl -Ls https://raw.githubusercontent.com/sdv77/youtube-downloader-tg-bot/refs/heads/main/setup.sh)`

Or you can use every possible method to copy script on your machine then make it executable `chmod +x setup.sh` and enjoy.
